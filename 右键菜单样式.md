一. 把 Win11 右键变回 Win10 样式；
改完后，不用再点 “显示更多选项”

- 按 Win+X → 选 终端 (管理员) 或 PowerShell (管理员)
- 粘贴下面的内容并回车：
```text
New-Item -Path "HKCU:\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" -Force
Set-ItemProperty -Path "HKCU:\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" -Name "(Default)" -Value ""
```
- 然后重启资源管理器 或 粘贴下面的内容并回车：
```text
Stop-Process -Name explorer -Force; Start-Process explorer
```

---

**二. 恢复 Win11 新菜单**

 方法一：管理员 PowerShell / 终端 一键恢复
- 按 Win+X → 选 终端 (管理员) 或 PowerShell (管理员)
- 复制下面两行，一次性粘贴回车：
```text
# 删除旧菜单的注册表项
reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f

# 重启资源管理器生效
taskkill /f /im explorer.exe; start explorer.exe
```
- 执行完，立刻恢复 Win11 新右键菜单

---

方法二：手动注册表删除
- Win+R → 输入 regedit → 回车（同意 UAC）
- 定位到：
```text
HKEY_CURRENT_USER\Software\Classes\CLSID
```
- 在 CLSID 下找到项：
`{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}`，
右键删掉它（连同里面的 InprocServer32 一起删）
- 重启资源管理器：
Ctrl+Shift+Esc → 找到 “Windows 资源管理器”→ 右键 “重新启动”。
